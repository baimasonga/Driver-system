# fleet_flutter

Flutter redesign of the Driver & Fleet Accountability Management System —
one codebase, two front-ends:

- **Web console** for fleet managers (`flutter run -d chrome`)
- **Android app** for drivers (`flutter run -d android`)

See the [repository root README](../README.md) for the full picture,
including the legacy React prototype this was redesigned from.

## Getting started

```bash
flutter pub get
flutter run -d chrome    # web console
flutter run -d android   # driver app
```
